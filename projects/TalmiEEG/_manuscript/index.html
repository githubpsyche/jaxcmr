The goal of this document is to document progress and note key decisions made during the development of a neurocognitive extension of eCMR using EEG data from Zarubin et al. (2020).

In the current project phase, the primary focus is on developing a simple prototype that demonstrates the feasibility of using item-level LPP signals to enhance eCMR’s predictions of the emotion-enhanced memory effect.

Future phases may explore alternative model specifications, fitting procedures, and evaluation metrics.

1 Data Preparation

Initially I prepared Single_Trial_Behavioural_and_EEG_Data_Z.csv from the lab drive, but missing study events and retrieval-success fields limited its value for model evaluation. I combined that partial LPP dataset with the more complete behavioral file All_Included_Subjects.csv. The file lacks Early/Late LPP scores but supplies full study-event records. For study events without LPP data, I imputed missing values using within-subject, within-trial averages for items in the same emotional category. Finally, I verified the completeness of this compiled dataset: 9 trials per participant, 20 study events per trial, each with a recall outcome.

It’s important to highlight that participants actually studied 22 items per trials, but the first two items from each trial were excluded from analysis to mitigate primacy effects. This is regrettable for model evaluation since CMR-style models often rely on early list items to establish context dynamics. Nonetheless, we will carefully navigate this limitation in the final version of our analyses.

Even more critically, the dataset does not record the order in which participants recalled items, only whether each item was recalled or not. This limitation requires custom-designed loss functions for model fitting even at the prototyping stage.

2.3 Recall Rate by LPP and Item-Condition

A successful neurally-informed extension of eCMR should not just identify emotional and neutral items, but provide additional information beyond these labels about how and whether emotional and neutral items will be recalled. We thus plot the relationship between LPP values and recall rate across item categories.

We observe that there are few observations at high (above 4.86) and low (below -3.25) LPP bins in our version of the analysis. In an early version of this analysis, I found that the relationship between LPP and recall rate was nonlinear. In fact, I was accidentally assigning a value 0 to subject-level bin value measurements when no observations were found in that bin. This led to the underestimation of recall rates in bins with few observations. With this issue corrected, a relatively positive trend can be observed between LPP values and recall rate, though variability in observation count across bins produces wide confidence intervals at high LPP amplitudes. An updated version of the analysis may seek to define bins so that the number of observations per bin is more balanced. Or we might just plot variables a different way.

2.4 LPP by Recall Status and Item-Category

To further validate the relationship between LPP and recall outcomes, I plotted LPP amplitudes separately for recalled and unrecalled items within each emotional category. Since we avoid binning LPP values in this analysis, this approach provides a cleaner view of how LPP varies with recall status across item types.

These plots broadly confirm that higher LPP amplitudes are associated with recalled items, but specifically for NEGATIVE items, with no clear relationship for NEUTRAL items. The pattern is also much clearer for Early LPP than Late LPP, supporting our focus on Early LPP for neurally-informed model extensions.

3 Loss Functions For Model Fitting

We face two main challenges when defining loss functions for fitting memory search models to the present dataset. First, memory search models like CMR and eCMR typically generate predictions about the order in which items are recalled, but the dataset only records whether each item was recalled or not. This limitation forces us to focus on a simpler question: does the composition of recalled items match what the model predicts?

Second, the first two studied items from each study list were excluded from the prepared dataset. This omission was intended to keep primacy effects from impacting analyses of the emotion-enhanced memory effect, but for fitting memory search models, it removes important information about how study lists were encoded and retrieved.

These two issues together mean that we cannot use standard likelihood-based loss functions that evaluate the probability of observed recall sequences under the model. For now, we ignore the excluded items, focusing on fitting models to predict the set of recalled and unrecalled items from each trial which we do have data for. To simplify model evaluation further, we also exclude evaluation of how models predict the total number of recalled items per trial. Neither CMR nor eCMR makes strong predictions about recall counts and addressing them requires additional parameters and mechanisms controlling recall termination. By excluding recall counts as a factor in loss functions and model evaluation, we reduce the complexity of the fitting process, focus squarely on neurally-augmented retrieved-context models’ ability to capture which items are recalled (rather than how many or in what order), clarifying the interpretability of model comparisons.

3.1 Set Likelihood

Without recall order data, we must evaluate models based on the probability they assign to the set of recalled items in each trial. The challenge is that memory search models are designed to generate predictions about the order of recalls, and multiple recall sequences are compatible with any given set of recalled items.

The most formally exact approach to overcome this challenge is to enumerate all possible recall sequences consistent with the observed set of recalled items and compute the probability of each sequence under the model. The probability of the observed data would then be the sum of the probabilities assigned to all compatible recall sequences. However, the number of possible recall orders increases factorially with the number of recalled items, making this approach computationally infeasible for all but the smallest datasets.

To sidestep this combinatorial explosion, a leading approach is to estimate the set likelihood using Monte Carlo sampling: drawing a constrained sample of recall sequences consistent with the observed recall set and averaging their model probabilities. This yields an unbiased estimate of the set likelihood but introduces sampling variability. By increasing the number of sampled sequences, the estimate can be made arbitrarily precise, albeit at the cost of increased computation time. Because reliable estimation would require a large number of samples per trial, we reserve this a complete examination of the approach for more advanced stages of the project.

3.2 Mean Squared Error (MSE) Over Summary Statistics

Given these constraints, we instead adopt a mean squared error (MSE) loss between an observed summary statistic and the same statistic generated by simulated data. This approach is widely used in evaluation of retrieved-context models, but is especially appropriate here given the limitations of our dataset.

The procedure is as follows:

  1. Compute a summary statistic from the experimental data.
  2. Simulate the model repeatedly using a given parameter set.
  3. Compute the same statistic from the simulated outcomes.
  4. Compute the mean squared difference between the observed and simulated statistics.

This MSE provides an interpretable measure of how well a given parameter set allows the model to reproduce the behavioral pattern of interest. In principle, if the model with the best-fitting parameters cannot reproduce the observed statistic well, we can conclude that the model is inadequate for explaining the data. While an MSE-based approach does not yield an unbiased estimate of the true likelihood or a maximum-likelihood estimate of parameters and discards trial-level prediction detail, it offers a practical, scalable, and transparent evaluation metric.

In initial evaluations, I experimented with different summary statistics to guide model fitting. First, I tried using the overall serial position curve (SPC) as the target statistic. This approach provides a straightforward test of whether the model captures overall primacy and recency trends across the list, ignoring differences between emotional and neutral items. However, this approach proved insufficient for evaluating the model’s ability to capture the emotion-enhanced memory effect, as it does not differentiate between item types.

We thus settled on using the category-SPC as the target statistic for model fitting. Here, recall probability is computed separately for negative and neutral items at each study position, allowing the model to be evaluated on its ability to reproduce both serial position effects and the emotion-enhanced memory effect simultaneously. Therefore, the category-SPC provides a more sensitive and relevant benchmark for assessing the performance of neurally-augmented eCMR models in this context.

4 Modeling

As a baseline, we begin evaluation with regular CMR, insensitive to item emotionality or LPP data. We evaluate this model to confirm that the emotion-enhanced memory effect cannot be captured without additional emotional-attention mechanisms.

Next, we implement a simplified version of eCMR that introduces emotional-attention mechanisms with minimal complexity. Rather than incorporate all parameters and mechanisms from eCMR, we seek to use only enough details to achieve adequate capture of our behavioral benchmark statistics.

From there, we will implement a neurally-informed extension of eCMR that uses event-level LPP measures as modulators of emotional-attentional strength. We will have two goals for this model. First, the model should outperform both baseline CMR and simplified eCMR at capturing the emotion-enhanced memory effect. Second, the model should adequately capture the relationship between event-level LPP values and recall probability for emotional and neutral items. Accomplishing these two goals together will produce a baseline neurocognitive model of the emotion-enhanced memory effect that can be further refined in future work using higher-quality datasets and more sophisticated model fitting procedures.

4.1 Simplified Pure-Attention eCMR

As an initial step beyond baseline CMR, we postulate that the emotion-enhanced memory effect arises entirely from differential allocation of attention during encoding, enhancing their context-to-item associations. We label this variant “pure-attention eCMR” to distinguish it from the complete eCMR specification.

The complete eCMR specification includes a separate emotional context and corresponding emotional feature-to-context and context-to-feature memories (eMFC and eMCF), with emotion-enhanced attention primarily modulating learning rates within these memory structures. They thus predict that emotion-enhanced attention primarily strengthens the association between items and emotion states during encoding. During recall, when emotional context is reinstated, it preferentially cues emotional items via these strengthened associations.

We exclude these dynamics at this stage, instead allowing emotional-attention to directly scale the strength of temporal context-to-item associations during encoding. We predict that this extension from baseline CMR alone is sufficient to capture the emotion-enhanced memory effect in our dataset, and will evaluate this prediction against more complex multi-component specifications in subsequent evaluations.

4.1.1 Parameter Sensitivity Tests

In Pure-Attention eCMR, an emotion_scale parameter scales the strength of context-to-item associations during encoding for emotional items.

To confirm that the model’s extra parameter can address the emotional category-spc, we fit the base model to the dataset then simulate pure-attention eCMR with different configurations of its emotion_scale parameter, plotting the emotional category-spc for each set of simulations.

These results show that scaling up the parameter increases recall rates for negative items while reducing recall rates for neutral items, building confidence that the parameter can improve the baseline model’s ability to address the emotion-enhanced memory effect. However, specific assumptions made by the model about issues like the interaction between primacy and arousal/emotional-attention as well as latent assumptions about effects on response order or initiation could still throw off model fitting.

4.1.3 Notes

While this text focus on just one specification, we considered two variants of pure-attention eCMR: one where emotional-attention scales context-to-item associations multiplicatively with primacy-based attention, and one where the two forms of attention combine additively. Preliminary simulations suggested that the additive variant better captured the emotion-enhanced memory effect without over-predicting recall rates for negative items in early study positions.

But both variants struggled to capture the emotion-enhanced memory effect adequately, both underestimating recall rates for negative items and overestimating recall rates for neutral items. The deficiency was biggest for negative items in later study positions, and for neutral items in study positions up through position 8.

The data exhibits considerable variability in recall rates for both item types across positions, and the model is maybe only able to capture the general trend beyond boosts in early and late positions characteristic of a U-shaped serial position effect.

Crucially, the pure-attention eCMR variant does not incorporate any mechanisms to capture the relationship between LPP amplitude and recall probability within each item category.

4.2 Simplified Two-Context eCMR

Another approach to simplifying eCMR is to retain its two-context structure but pare down the number of free parameters and mechanisms. In this approach, we retain the emotional and temporal contexts, but make the key simplification of fixing the integration rate of emotional context to 1.0. This means that the emotional context at each step of either encoding or retrieval is determined entirely by the most recently processed item, making emotional feature-to-context memory (eMFC) effectively redundant since emotional context can be updated directly from item features.

The emotional context-to-feature memory (eMCF) is retained and represented as a vector where cells corresponding to emotional items hold a value scaled by a emotion_scale parameter, while cells corresponding to neutral items hold a value of 0.

An additional parameter sometimes used in eCMR to scale the relative contribution of temporal vs emotional context during retrieval is redundant under this specification, since the emotion_scale parameter already modulates that balance. With specialized drift rates and feature-to-context memory learning rates also excluded, this simplified two-layer eCMR addresses the emotion-enhanced memory effect with still just one additional free parameter beyond those specified in baseline CMR.

Whereas the pure-attention eCMR variant postulates that emotional-attention modulates context-to-item associations directly, the two-context eCMR variant predicts that the boost in recall for emotional items arises from their ability to reinstate emotional context, which then preferentially cues other emotional items. We can’t directly evaluate this mechanism with the present dataset since we lack recall order data. Nonetheless, the model may still make different predictions about overall recall rates for emotional and neutral items than the pure-attention variant and at minimum can be evaluated for its ability to capture the emotion-enhanced memory effect.

4.2.1 Parameter Sensitivity Tests

Compared to pure-attention eCMR, the two-context eCMR variant shows a similar sensitivity to changes in the emotional-attention scaling parameter, with increases in the parameter boosting recall rates for negative items while reducing recall rates for neutral items. The Two-Context eCMR variant appears to produce a bigger boost to recall rates for the first and last study positions for negative items, while the pure-attention variant seems to apply some kind of soft ceiling on recall rates for negative items in the first and last study positions.

4.3 Neurally Informed Extension

We now turn to exploring how a neural signature known to carry information about whether negative items will be recalled or not can further augment model performance. Our focus for the moment is an initial prototype that efficiently demonstrates that such improvement using present data is possible. Downstream work can attempt to identify the best approach.

4.3.1 Emotion Enhanced LPP

Before settling on a specification, here we seek to clarify what we know about the LPP signal that we’ll be using to extend eCMR:

  • Early LPP values are consistently higher for NEGATIVE items than for NEUTRAL items.
  • For NEGATIVE items, LPP values are consistently higher for RECALLED items
  • For NEUTRAL ITEMS, LPP values are do not consistently index whether an item will be recalled

5 Side Issues

5.1 Do arousal and serial position additively or interactively influence memory strength?

The specification for eCMR seems to configure a multiplicative relationship between emotional-attention and primacy-attention. I think that leads to the prediction that the recall rate gap for emotional vs neutral items depends on the compared items’ study positions: higher for items studied near the beginning of a list than elsewhere.

Here’s a worked example:

If the primacy-attention value at the first study position is 10 and the list-wide emotional attention scale is 3, then the total attentional boost at that study position is 10*3=30 for emotional items but just 10 for neutral items. So the difference in attention for emotional vs neutral items is 20.

At a later study position, the primacy attention value falls to 1 and the emotional-attention scale is still 3. This makes the total attentional boost at that study position 1*3=3 for emotional items but just 1 for neutral items. The difference in attention for emotional vs neutral items is just 2.

That’s ten times lower than the emotional vs neutral boost at study position 1! So according to this specification, the attentional boost that drives the emotion-enhanced memory effect is stronger for early study positions than late study positions.

The present data and model evaluation pipeline present only a limited opportunity to examine whether emotion and primacy independently/additively or interactively/multiplicatively configure emotional context-to-item memory strengths. Here we draw separate plots for data and each model variant.

These simulations suggest that primacy and emotionality independently rather than interactively modulate memory strength. The gap between the two variants seems localized to their predictions about recall rates for negative items in early study positions. The multiplicative model tends to predict a much higher recall rate for these items than observed in the data, while the additive model produces no such discrepancy. We thus use the additive model for simulations going forward, and propose pursuing a paper focused on this distinction as a future project.

5.2 Improving Trade-Off Between Recency and Primacy

Baseline and eCMR fits seem to give low recall rates to the last few study positions in our serial position curves. One possible reason is that CMR lacks a mechanism to eliminate recency effects without boosting the strength of the primacy effect (through start-of-list context reinstatement). Since the data exhibits neither a strong recency nor a strong primacy effect, the model struggles to capture it.

On the other hand, it could be that eCMR’s complete specification is necessary to avoid this down-modulation of the serial position effect. However, since eCMR’s eMCF only affects recall rates for emotional items, I think it unlikely to help address the down-modulation observed for the neutral-SPC.

A common approach in retrieved-context models to attenuate the recency effect is to introduce a delay drift rate following the study phase, causing context to drift toward an out-of-list context before retrieval begins. The rationale is that by shifting the contextual cue away from its final study state, recently studied items will receive less contextual overlap and hence less retrieval advantage.

However, this mechanism generally fails to balance primacy and recency because it does not alter the relative similarity between the contextual cue and the study-phase contexts of individual items. Context moves equally far from all studied items, preserving their relative strengths as retrieval cues.

To overcome this limitation, I tested a modified approach where post-study context drifts not toward an external or out-of-list context, but toward all in-list contexts, where the input vector is essentially a ones mask over context units that are linked to any studied item. This adjustment removes temporal directionality from the final cue, effectively making recall initiation reflect just the differences in relative strength across context-to-item associations.

This didn’t help either, so I’m leaving this problem for later..
