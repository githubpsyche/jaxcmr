Remembering for Reasons: Toward a Neurocognitive Model of Goal-Congruent and Incongruent Memory Search

The broad objective of memory is to relate past experience to ongoing needs.
However, external cues are often too sparse or misleading to guide retrieval unaided.
Humans bridge this gap by initiating an iterative memory search, updating self-generated cues from ongoing context and retrieval outputs to probe stored experience.
This internally guided search underpins adaptive behavior, but current theories still tackle directed recall task-by-task, tuning assumptions for each paradigm in isolation.
That fragmentation leaves us without a principled account of why identical study experiences yield different recall patterns when goals shift.
Nor does it equip us to flexibly explain when control fails, as when emotionally intense memories intrude unbidden.

This project addresses that gap by treating retrieval goals as explicit, manipulable variables within a computational model of memory search.
Current frameworks absorb goal-related variance into task-specific parameters, fitting each paradigm in isolation.
By representing goals directly, the model predicts how recall should change when goals change, without refitting.
The same framework accommodates control failure: when a memory's accessibility outweighs goal-based modulation, it can surface despite intentions, producing intrusions.
This challenges a longstanding assumption in memory research: that accessibility and intentional control can be measured and modeled separately.
The proposed experiments use behavioral recall patterns and electrophysiological markers to adjudicate between candidate mechanisms specifying how goals configure the retrieval process.
In turn, the model lays groundwork for a unified account of goal-directed remembering across laboratory tasks and clinical populations where such failures are central.

Our approach is to collect behavioral recall data and electrophysiological recordings while participants perform directed memory-search tasks that vary retrieval instruction, instruction timing, and item salience.
The model builds on retrieved-context frameworks (Howard & Kahana, 2002; Polyn et al., 2009), where gradually changing temporal context is bound to each episodic trace and later serves as the cue for recall.
That temporal-targeting mechanism explains benchmark phenomena like lag-recency curves and latency distributions, but treats recall patterns as direct reflections of accessibility, with no explicit role for goals.
CMR2 (Lohnas et al., 2015) introduced a post-retrieval monitoring process to address this limitation, explaining the suppression of out-of-list intrusions through an assessed match between items' retrieved context and the ongoing context.
We seek to generalize this monitoring architecture from list-membership discrimination to arbitrary goal definitions, representing goals as explicit states that can specify reward priority, category targets, serial position, or suppression instructions.
The proposed experiments adjudicate between candidate mechanisms for how goals configure retrieval (shaping cue composition, modulating competition dynamics, or filtering outputs post-selection), using behavioral patterns and electrophysiological markers to constrain model selection.
We will deploy this approach to accomplish the following specific aims:

**Aim 1: Formalize and test goal-conditioned control within episodic memory search.** 
We will generalize CMR2's monitoring architecture by representing retrieval goals as explicit states that can specify arbitrary task requirements (list membership, serial position, category, reward priority). 
We will fit competing model variants to existing datasets in which the same participants perform multiple directed-recall tasks (free, serial, categorized, reward-prioritized), allowing us to test whether goals configure retrieval by shaping cue composition, modulating competition dynamics, or filtering outputs post-selection. 
Simulations will evaluate each mechanism's ability to capture within-subject, instruction-dependent shifts in recall order, latency, and intrusion patterns without task-specific refitting.

**Aim 2: Experimentally dissociate goal effects at encoding versus retrieval.** In new directed-recall experiments, we will manipulate retrieval instruction (free, serial, categorized, reward-maximizing), instruction timing (pre- vs post-study), and practice history while recording behavior and EEG. The timing manipulation is diagnostic. 
If goals primarily shape cue composition, pre-study instructions should produce larger effects than post-study instructions; if goals modulate competition or filtering at retrieval, instruction timing should matter less. 
Late positive potentials at study will index goal-based encoding prioritization; encoding-retrieval similarity at test will index goal-dependent reinstatement. 
We will compare how well each mechanism accounts for the joint pattern of behavioral and neural data across timing conditions.

**Aim 3: Test how emotional intensity interacts with goal-configured control.** We will extend the Aim 2 designs by varying item salience (reward-tagged, emotional, or neutral) while holding instruction and timing fixed. This manipulation directly tests the control-failure mechanism: emotionally intense items should show elevated accessibility that can override goal-based modulation, producing intrusions when accessibility outweighs goal fit. Behavioral and EEG data will evaluate whether goals and arousal engage overlapping or distinct control processes, and clarify how arousing experiences can alternatively focus retrieval (when goal-congruent) or intrude upon it (when goal-incongruent). This aim lays groundwork for understanding intrusive memories in clinical populations.
