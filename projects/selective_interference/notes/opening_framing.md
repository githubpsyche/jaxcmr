## Focal problem

The opening must do five things in sequence:

1. **P1** establishes a system-level tension between durability and selectivity in episodic remembering. (Developed in `p1_framing.md`.)
2. **P2** introduces the trauma-film paradigm and the dissociation between involuntary and voluntary retrieval.
3. **P3** names the selective interference phenomenon — a visuospatial task reduces intrusions while sparing voluntary memory.
4. **P4** introduces the dual representation account (DRA) as the dominant explanation of the selective interference effect.
5. **P5** states the retrieved-context theory (RCT) thesis as an alternative.

**Structural rationale (5 paragraphs vs. repfr's 3):** In repfr, the paradigm IS the diagnostic — repeated-item lists directly measure integration/differentiation. In selintfr, the paradigm (trauma-film) and the diagnostic phenomenon (selective interference) are separable, requiring separate paragraphs (P2, P3). Additionally, the theory under scrutiny (DRA) and the proposed alternative (RCT) each require a full paragraph to convey their explanatory logic — repfr could combine these because the theory under scrutiny (standard CMR) was already familiar to the target readership.

## Constraints

### Prose constraints (apply everywhere)

Carried forward from `p1_framing.md` where they originated.

| | Constraint |
|---|---|
| C11 | **One sentence, one job.** No overextension with em-dashes, semicolons, or colons to express multiple ideas. These punctuation marks can still be used if they don't violate the principle. |
| C12 | **Active voice.** Passive and stative copula constructions weaken sentences. Prefer active verbs. Functional register ("preserve," "return," "surface") naturally produces active voice; mechanistic register ("encoded," "gated," "reactivated") invites passives. |
| C15 | **Conciseness.** Target 4–5 sentences per paragraph. |
| C16 | **Accessible register.** Avoid gratuitous jargon. Functional language is acceptable when precise. |
| C17 | **Rhetorical framing must be earned.** A sentence can only invoke a framing concept (e.g., "puzzle," "paradox," "challenge") if that concept has been established by prior sentences. Unearned framing presupposes a reader state that doesn't exist. |

### Arc constraints (apply across paragraphs)

| | Constraint |
|---|---|
| A1 | **Linking phrases must connect.** Each paragraph's opening phrase must refer to a concept actually present in the prior paragraph. |
| A2 | **Vocabulary consistency.** Key terms should have one meaning across the opening. If a word is used in two senses, the reader will stumble. |
| A3 | **Forward pull.** Each paragraph should leave the reader wanting what the next paragraph delivers. |
| A4 | **Structurally distinct from repfr.** Both papers may be read by the same reviewers. Different sentence structures and rhetorical moves. |
| A5 | **Hedging calibrated to epistemic status.** Empirical claims hedged where findings are heterogeneous. Theoretical framing uses "in this account" / "suggest" rather than "we show." The opening frames; the paper argues. |
| A6 | **Standalone self-sufficiency.** The opening must function as a complete, circulatable draft without subsections. Every concept needed to evaluate the thesis must be introduced within P1–P5. No sentence should presuppose content from a later section that doesn't yet exist. |

### Per-paragraph constraints

**P1** — See `p1_framing.md` for the full constraint set (C1–C17). Key constraints summarized:

| | Constraint |

| P1-C10 | No implication that trauma is beneficial. |
| P1-C13 | Name the problem, not the mechanism. Relaxed for P4–P5 (which must name mechanisms at functional level). |
| P1-C14 | Traumatic memory ≠ pathological memory. |

**P2** — The paradigm.

| | Constraint |
|---|---|
| P2-C1 | Introduce the trauma-film paradigm as a concrete instantiation of P1's tension. |
| P2-C2 | Establish the two retrieval modes (involuntary and voluntary) as a dissociation, not separate systems. |
| P2-C3 | Hedging: the dissociation is empirical ("can be weak," "can come apart"), not absolute. |

**P3** — The phenomenon.

| | Constraint |
|---|---|
| P3-C1 | Name the selective interference effect. |
| P3-C2 | Cover both immediate and delayed (post-reminder) evidence. |
| P3-C3 | The final sentence should sharpen the puzzle — same experience, different retrieval outcomes. |

**P4** — The dual representation account.

| | Constraint |
|---|---|
| P4-C1 | DRA introduced as "a prominent interpretation," not "the prevailing view." |
| P4-C2 | DRA presented as an account of selective interference, not a general primer. The paragraph answers: how does DRA explain the phenomenon P3 described? |
| P4-C3 | Two traces named at functional level (what they do — support intrusions vs deliberate recall), not encoding-process level (how they are formed). Subsection's job. |
| P4-C4 | Covers both immediate (resource competition during consolidation) and delayed (reconsolidation) mechanisms. |
| P4-C5 | Final sentence names what DRA commits to, using the same vocabulary P5 S5 will challenge (separate traces, modality-specific disruption, reconsolidation window). Sets up a clean handoff. |

**P5** — The RCT thesis.

| | Constraint |
|---|---|
| P5-C1 | Thesis scoped as parsimony argument ("without requiring ... as modeling prerequisites"), not blanket dismissal. |
| P5-C2 | Names all three DRA assumptions the paper challenges: separate traces, reconsolidation windows, modality-specific disruption. |
| P5-C3 | Names both mechanisms: context overlap (interference) and retrieval control (protection). |
| P5-C4 | Standalone scaffolding — one sentence characterizing the general framework before applying it. The opening circulates before subsections are written; a reader unfamiliar with RCT needs a bridge from the framework name to its application. |
| P5-C5 | Closing motivation — P5 must close with a statement of why the alternative matters beyond parsimony: theoretical unification, divergent predictions, or clinical implications. |

**P1:**
> Traumatic experience highlights a trade-off between durability and selectivity in episodic remembering. Memory should retain highly consequential episodes so critical details stay accessible long into the future. At the same time, adaptive retrieval is selective: what comes to mind should normally depend on current cues, goals, and safety. **P2:**

**P3:**
> A sharper dissociation emerges when a brief visuospatial task follows the film or a later film reminder. Across studies, participants who play Tetris or perform a similarly visually-engaging task report fewer intrusive memories over the following days. Voluntary memory for the same material generally shows little measureable impact even when retrieval conditions are matched. Such interventions are effective even after prompting reinstatement of real-life traumatic memories, reducing intrusions while preserving deliberate recall for life experiences such as motor vehicle accidents, traumatic childbirth following emergency caesarean section, and war-related trauma in refugees. A post-encoding or post-reminder manipulation appears to alter one mode of retrieval more than the other, even when both draw on the same recent experience.

**P4:**
> A prominent interpretation attributes this selectivity to separable memory representations. Under this dual representation account, traumatic experience gives rise to distinct sensory and contextual traces, and a post-encoding task preferentially disrupts the sensory trace while leaving the contextual trace (and the voluntary memory it supports) relatively intact. Here we develop an alternative within a unitary retrieved-context framework and demonstrate it in a formal model. In this account, the degree of contextual overlap between the interference task and the original event governs retrieval competition, while the amount of control available at test governs which memories are expressed. The dissociation between involuntary and voluntary recall emerges from these interacting constraints without requiring separate traces as a modeling prerequisite.

**Problems that motivated the split:**

1. **P4 mixed DRA exposition and RCT thesis in one paragraph.** S1–S2 introduce DRA; S3–S5 state the RCT thesis. Neither gets enough space to convey its explanatory logic.
2. **DRA got only 2 sentences.** The reader cannot evaluate the alternative (RCT) without understanding what the dominant account actually claims — how it explains the selective interference effect, not just that it posits "separable memory representations."
3. **No reconsolidation coverage.** DRA's account of delayed interventions (reminder → reconsolidation window → restabilization disruption) was absent. The paper challenges this assumption, so it must be introduced.
4. **Thesis narrower than the paper's argument.** "Without requiring separate traces as a modeling prerequisite" names one of three DRA assumptions the paper challenges. The other two (reconsolidation windows, modality-specific disruption) were deferred to the body.
5. **Citation formatting bug.** "representations. [@brewin1996; @brewin2014]." — period before citation created a sentence fragment.

**P4:**
> A prominent interpretation attributes this selectivity to separable memory representations. Under the dual-representation account, traumatic experience produces distinct sensory and contextual memory traces — the first supporting involuntary re-experiencing, the second supporting deliberate recall. A visuospatial interference task competes for the perceptual processing resources needed to consolidate the sensory trace, reducing later intrusions while leaving the contextual trace and the voluntary memory it supports relatively intact. When the intervention follows a reminder at longer delays, the account invokes reconsolidation: the reminder reactivates the sensory trace, and the interfering task disrupts its restabilization during a time-limited window. The account thus ties selective interference to separate memory representations, modality-specific disruption of one trace, and — for delayed effects — a discrete reconsolidation window.

**P5:**
> Here we develop an alternative within a unitary retrieved-context framework and demonstrate it through simulations of a formal model. The framework holds that items are bound to a continuously evolving temporal context that later serves as the primary retrieval cue. In this account, the original event and a later interference task deposit traces in this shared context, and the degree of overlap between their contexts governs retrieval competition. Retrieval control mechanisms available during deliberate recall, reinstating the event's context and sharpening competition among candidates, can preserve the target's advantage over competitors. The dissociation between involuntary and voluntary recall emerges from these interacting constraints without requiring separate traces, reconsolidation windows, or modality-specific disruption as modeling prerequisites. By grounding selective interference in general principles of temporal context and retrieval competition, the account connects a clinical phenomenon to the broader episodic memory literature and generates predictions that diverge from the dual-representation account and can clarify intervention designs.

## Open questions

1. **P3-C1 (selective interference effect unnamed) — resolved.** Named in S5: "This selective interference effect deepens the dissociation." S6 characterizes the effect as its own sentence: "A post-encoding or post-reminder manipulation appears to alter one mode of retrieval more than the other, even when both draw on the same recent experience." S5/S6 split gives the naming its own moment (C11 improvement) at the cost of a 6th sentence in P3 (C15 borderline). "Deepens the dissociation" connects to P2's "dissociate" and P3's "A sharper dissociation" — grounded vocabulary chain. Earlier "sharpens the puzzle" presupposed a puzzle the reader didn't yet have (C17 violation).

2. **P3 S4 (clinical translation sentence).** Kept by decision, but it's the heaviest sentence in the opening (lists three clinical applications). C11 borderline. If the sentence is cut later, the delayed-intervention evidence moves entirely to P4 S4 ("when the intervention follows a reminder at longer delays"), which may be sufficient.

3. **P4 S2 em-dash.** Same pattern flagged repeatedly in P1 drafting. Defensible here because "sensory and contextual memory traces" is uninformative without the functional elaboration. But if a colon-free alternative emerges naturally, it would be cleaner.

4. **P5 S3 length.** (Was S2 before bridge sentence added.) Joins two clauses with "and." Could be split into: "In this account, the original event and a later interference task deposit traces in this shared context. The degree of overlap between their contexts governs retrieval competition." Trade-off: splitting costs a sentence (C15, 5→6) but improves C11.

5. **P4→P5 link.** "Here we develop an alternative" is an implicit link — the reader infers "alternative to what P4 just described." Explicit links have been preferred elsewhere (A1). An option: "Here we develop an alternative to this account..." but that adds words without adding information. Accepted as implicit.

6. **P5 S2 "are bound" passive.** "Items are bound to a continuously evolving temporal context" uses passive voice. Defensible: the agent is the encoding process, which hasn't been named yet and naming it would bloat the sentence. Active alternative ("the framework binds items to...") anthropomorphizes the framework. Accepted as functional passive.
