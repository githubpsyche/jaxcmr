## Recap of Last Meeting: Key Resolutions

- Primary empirical target:
  - Model the selective interference effect in the Lau-Zhu, Henson & Holmes trauma-film + Tetris paradigm.
  - Focus initially on the 2019 JEP:G and 2021 CPS papers.
- Modelling framework:
  - Use a retrieved-context model (CMR/eCMR-family) as the starting point.
  - Treat this as the unitary/single-trace baseline against which DRA-style variants can be contrasted.
- Empirical anchor:
  - Treat the 2021 CPS experiment as the preferred modelling target for the interference dissociation, because:
    - Intrusions (vigilance-intrusion) and recognition are tested on the same day, in the same context.
    - They use the same film vs foil stills as cues and the same digit go/no-go structure during cue presentation.
  - Use 2019 Experiment 2 as a secondary check (Day 8 vigilance-intrusion vs uncued recognition) once the basic pipeline is working.
- Data constraints:
  - Accept that we only have summary-level behavioural data from these papers.
  - Treat this phase as a proof-of-concept: simulate the published designs under pre-existing parameter sets rather than fitting parameters directly to these datasets.

::: notes
This is my understanding of what we converged on:
- We are not trying to rebuild the entire Tetris/trauma literature in CMR.
- We are trying to see whether a retrieved-context model, in something like its current form, can plausibly reproduce the pattern: reminder + Tetris reduces intrusions but leaves voluntary memory intact.
- The 2021 experiment is the best single test of that because it controls away cue and task differences between intrusions and recognition.
- Given the absence of trial-level data, we are aiming for a principled proof of concept rather than full parameter fitting.

## Present Research Plan: How We Will Use CMR

- Conceptual role of CMR/eCMR:
  - Provide a unitary retrieved-context account of both intrusions and voluntary memory.
  - Use that as the reference point for evaluating whether DRA-style dual-trace commitments are actually necessary to explain the data.
- Mapping the experiments into CMR terms:
  - Trauma film: sequence of item events whose features are bound to a drifting context state.
  - 30-minute delay: continued context drift with no item encoding.
  - Reminder + Tetris vs reminder only:
    - Reminder: brief reactivation of film-related context and items.
    - Tetris: additional manipulation of the visuospatial channel and/or context drift during this reactivated window.
  - Vigilance-intrusion task:
    - Repeated presentation of neutral cues (film or foil stills) while context evolves and occasionally provokes intrusive retrievals of film hotspots.
  - Recognition task:
    - Cued or uncued old/new decisions driven by similarity between the current cue-induced context and item traces from the film vs foils.
- Modelling goals for this phase:
  - Show whether a reasonable CMR-style model can produce:
    - A large reduction in simulated intrusions in the interference group.
    - Little or no change in simulated recognition performance under matched retrieval conditions.
  - Identify what additional assumptions (if any) are required to get there:
    - Channel-specific drift or interference.
    - Different retrieval thresholds or gating rules for spontaneous vs deliberate retrieval.
    - DRA-inspired changes in sensory vs contextual binding.

::: notes
The key thing I want to make explicit is that I am not starting from a bespoke "dual representation model". I am starting from a standard retrieved-context model and asking:
- Can this architecture reproduce the selective interference pattern at all, given plausible parameters and a plausible implementation of the Tetris manipulation?
- If not, what specific changes are required, and how do those changes map onto dual-representation commitments?

This gives us a clean way to compare:
- A unitary, CMR-like account where interference acts on context dynamics and retrieval competition.
- A dual-representation account where affect selectively strengthens sensory/affect traces and weakens links to temporal context.

Questions:
- Are there particular DRA claims you want baked into the first model, or should we keep the initial implementation as close as possible to vanilla CMR/eCMR and only add DRA structure once we see what that baseline can and cannot do?
- Are there any aspects of the trauma film paradigm (e.g., emotion, imagery vividness) that you regard as non-negotiable to capture qualitatively in this first pass?
:::

## Concrete Short-Term Plan: Implementation Steps

- Step 1: Choose and fix a base CMR implementation and parameter set
  - Start from an existing CMR/eCMR implementation that already fits list-learning data (free recall, possibly recognition).
  - Select one or two parameter sets that give realistic temporal contiguity and item memory without any trauma-specific tweaks.
- Step 2: Abstract the 2021 CPS experiment into a simulation script
  - Represent the trauma film as a sequence of item events with designated hotspot items.
  - Implement the 30-minute delay, reminder cues, and two post-film conditions:
    - Reminder + Tetris: add an extra phase where context drift and/or visuospatial interference is increased during reactivation.
    - Reminder only: same reminder but no added interference.
  - Implement the Day 1 vigilance-intrusion task:
    - Alternate digit go/no-go trials with background cues (film vs foil stills).
    - Define intrusions as high-activation film hotspots crossing a threshold in response to film cues (and occasionally foil cues).
  - Implement the Day 1 modified recognition task:
    - Use the same cue set as in the intrusion task (film vs foil stills with go/no-go digits).
    - Follow cues by noisy film or foil targets and compute old/new decisions based on context-driven similarity.
- Step 3: Define observables and analysis pipeline
  - Vigilance-intrusion:
    - Simulated metric: mean number of hotspot intrusions per condition (film-cue vs foil-cue) and per group (interference vs control).
    - Match the published analysis: group means and effect sizes for early intrusions.
  - Recognition:
    - Simulated metric: hit and false-alarm rates for film vs foil targets, broken down by cue type.
    - Compute the same accuracy index (hits minus false alarm rate) as in the paper and compare interference vs control.
  - Aim: reproduce the headline dissociation pattern, not every detail of the distributions.
- Step 4: Explore minimal variants if baseline CMR fails to show a dissociation
  - If standard parameters give either no effect or symmetric impairment of intrusions and recognition, systematically introduce:
    - Stronger context drift or reduced reinstatement during the interference window.
    - Different retrieval thresholds or competition rules for spontaneous vs deliberate recall.
    - DRA-like manipulations where sensory features are less dependent on temporal context.
  - Track which changes move the model toward the observed pattern and what they imply theoretically.
- Step 5: Prepare summary outputs for discussion
  - For the next meeting, aim to bring:
    - Schematic of the simulation timeline with CMR operations annotated.
    - Plots mirroring the key figures: intrusions by group and cue type; recognition accuracy by group and cue type.
    - A short list of candidate mechanisms that did or did not produce a selective interference pattern.

::: notes
This is the concrete work plan I am following between now and the next couple of meetings.

Important constraints and assumptions:
- I am reusing parameter sets from richer list-learning data rather than fitting directly to the trauma-film curves, at least initially.
- I am treating intrusions as thresholded spontaneous retrievals of designated hotspot items in response to cues, which is a simplification but keeps the link to CMR transparent.
- I am focusing first on the 2021 experiment because it gives us matched intrusion and recognition tasks; 2019 Exp 2 can be added once this pipeline is in place by swapping in the Day 8 intrusion and uncued recognition structures.

Questions I would like explicit feedback on:

- Does the definition of "intrusion" I am using in the model (hotspot retrieval above threshold during film-cue trials) align with how you want to interpret the vigilance-intrusion data?
