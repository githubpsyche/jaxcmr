## Prerequisites

Before starting these tutorials:

1. Install jaxcmr (`pip install jaxcmr`)
2. Review [CMR parameters](../concepts/parameters.qmd)

## Working with the Tutorials

These tutorials are based on Jupyter notebooks in the `templates/` directory. You can:

1. **Read online**: Follow the rendered tutorials here
2. **Run interactively**: Open the notebooks in Jupyter and modify parameters
3. **Use as templates**: Copy and adapt for your own experiments

```bash
# Open the fitting template
jupyter notebook templates/fitting.ipynb
```
